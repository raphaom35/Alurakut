## Features

- ⚛️ **React** — A lib that provides a way to create web app
- 💅 **Styled Components** — styled-components
- 💖 **Lint** — ESlint/Prettier/Editor Config

## Getting started

1. Clone this repo using `git clone https://github.com/raphaom35/alurakut.git`
2. Move yourself to the appropriate directory: `cd alurakut`<br />
3. Run `yarn` to install dependencies<br />
4. Run: `yarn dev`
